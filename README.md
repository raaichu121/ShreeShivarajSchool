## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or bun package manager

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/raaichu121/ShreeShivarajSchool.git
   cd shree-school-ai-main
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   # or
   bun run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the website.
